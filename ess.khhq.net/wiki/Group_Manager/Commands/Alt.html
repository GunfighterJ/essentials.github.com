User

User Management

  • /manadd: Set/Add/Remove a group to a player or group.
    • /<command> <player> <group> : Sets the main group
    • /<command> <player> +<group> : Add a subgroup
    • /<command> <player> -<group> : Remove a subgroup
    • /<command> @<group>  : Creates a group
    • /<command> @<group1> +<group2> : Add a group to inherit
    • /<command> @<group1> -<group2> : Remove an inheritance
    • groupmanager.manadd
    • groupmanager.manaddsub
    • groupmanager.mandelsub
    • groupmanager.mangadd
    • groupmanager.mangaddi
    • groupmanager.mangdeli
  • /mandel: Remove user or group from the config (Will become default)
    • /<command> <player>
    • /<command> @<group>
    • groupmanager.mandel
    • groupmanager.mangdel
  • /manpromote: Promote a player in the same heritage line to a higher rank.
    • /<command> <player> <group>
    • groupmanager.manpromote
  • /mandemote: Demote a player in the same heritage line to a lower rank.
    • /<command> <player> <group>
    • groupmanager.mandemote

info

  • /manwhois: List user information and overrides.
    • /<command> <player>
    • /<command> <player> <permission>
    • /<command> <player> <variable>
    • /<command> @<group>
    • /<command> @<group> <permission>
    • /<command> @<group> <variable>
    • groupmanager.manwhois
    • groupmanager.mancheckp
    • groupmanager.mancheckv
  • /listgroups: List the groups available.
    • /<command>
    • groupmanager.listgroups

permissions

  • /manaddp: Add permission directly to a player or to a group.
    • /<command> <player> <permission>
      • /<command> @<group> <permission>
    • groupmanager.manaddp
    • groupmanager.mangaddp
  • /mandelp: Removes permission directly from a player or from a group.
    • /<command> <player> <permission>
    • /<command> @<group> <permission>
    • groupmanager.mandelp
    • groupmanager.mangaddp


variables

  • /manaddv: Add, or replaces, a variable to a user/group (like prefix or suffix).
    • /<command> <user> <variable> <value>
    • /<command> @<group> <variable> <value>
    • groupmanager.manaddv
    • groupmanager.mangaddv
  • /mandelv: Remove a variable from a user.
    • /<command> <user> <variable>
    • /<command> @<group> <variable>
    • groupmanager.mandelv
    • groupmanager.mangdelv

Utility Commands

  • /mansave: Save all permissions on file.
    • /<command>
    • groupmanager.mansave
  • /manload: Reload current world and config.yml. Or load given world.
    • /<command> [world]
    • groupmanager.manload
  • /mantoggle: Toggle on/off a setting.
    • /<command> save : Autosaving
    • /<command> validate : Name validation
    • groupmanager.mantogglesave
    • groupmanager.mantogglevalidate
  • /manworld: Selects or displays the current world
    • /<command>
    • /<command> set
    • /<command> set <world>
    • groupmanager.manworld
    • groupmanager.manselect
