Command Reference/Perm

From Essentials

Jump to: navigation, search

Essentials is made up of multiple components, you must place each jar that you intend to use in your Minecraft/Bukkit plugins folder. You will generally not need/want all components, ignore the commands & permissions from the modules you haven't installed.

Most of the following commands are located in the core Essentials.jar file.

  • Commands in the 'Spawn' category require EssentialsSpawn.jar
  • Commands in the 'XMPP' category require EssentialsXMPP.jar

Permissions aren't automatically inclusive, you will need to give both 'essentials.warp' and either 'essentials.warp.mall' or 'essentials.warp.*' to allow use of "/warp mall".
